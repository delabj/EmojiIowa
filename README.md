# EmojiIowa


Emoji Iowa is a simple twitter bot hosted using Google App Scripts to that uses openweathermaps api and the twitter api to automatically tweet a weather map of Iowa at the current time. The bot lives on Twitter [@EmojiIowa](https://twitter.com/EmojiIowa)
